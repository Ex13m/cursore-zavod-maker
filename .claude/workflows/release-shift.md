# Workflow: полная смена завода вручную

Когда владелец просит «прогони полный цикл смены»:

1. **Тренды**: `node factory/trends.js` — проверь notes (какие палитры в фаворе).
2. **Производство**: `node factory/produce.js` (идемпотентно за сегодня).
3. **ОТК-сводка**: прочитай дроп, перечисли тревоги; критичные (пустые effects,
   unknown type) — чини сразу, косметические (contrast) — оставь владельцу.
4. **Коммит**: `git add factory/data && git commit -m "factory: drop <дата>"` + push
   (Netlify пересоберёт портал автоматически).
5. **Письмо**: `node factory/notify.js` (без RESEND-ключа — ссылка в stdout).
6. Отчитайся: сколько выпущено, сколько тревог, ссылка на пульт.
