# Правила зоны portal/

- Дизайн-токены — только из `src/style.css :root`. Цвет = статус
  (RUN/QUEUED/ALARM/INFO), остальное — графитовые серые.
- Не анимировать интерфейс: движется только сам курсор (правило high-performance HMI).
- Бюджет: ≤ 100 KB JS в сборке. Без фреймворков и рантайм-зависимостей.
- Данные читаются из `/data/**` (статика), действия — `/api/**`.
- Контракт API одинаков для dev (factory/devserver.js) и прод (netlify/functions):
  меняешь эндпоинт — меняй в обоих местах.
