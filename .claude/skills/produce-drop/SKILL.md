# Выпуск дропа

1. Обнови тренды и произведи дроп:
   ```bash
   node factory/trends.js
   node factory/produce.js ${DATE:-} ${FORCE:+--force}
   ```
   (`--force` — только если явно просят пересоздать; иначе идемпотентный skip.)
2. Прочитай `factory/data/drops/<дата>.json` и дай сводку: количество, имена,
   палитры, тревоги ОТК (каждую тревогу — с объяснением).
3. Если запущен dev (`npm run dev`) — напомни, что дроп уже виден на
   http://localhost:5173 (вкладка КОНВЕЙЕР).
4. Коммить дроп сообщением `factory: drop <дата>` — только если просили закоммитить.
