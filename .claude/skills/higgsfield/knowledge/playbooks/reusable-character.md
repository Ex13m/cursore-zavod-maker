# Рецепт: переиспользуемый персонаж / аватар / «моё лицо»

**Задача:** один и тот же персонаж в разных генерациях (кадры, ролики, реклама).

**Развилка (спросить пользователя, если неясно):**
- **Soul** — верность одному лицу, «цифровой двойник», 5–20 фото, ~10 мин обучения.
  Один `soul_id` на генерацию. Работает ТОЛЬКО с `soul_2` / `soul_cinema_studio`.
- **Elements** — мгновенно, из 1 снимка, можно НЕСКОЛЬКО субъектов в кадре
  («я + друг»), работает с Nano Banana Pro/2, Seedream, Kling 3.0, Seedance 2.0.

Сигналы за Soul: «обучи»/«двойник»/«идентичность»/5+ фото одного человека.
Форсить Elements: >1 персонажа в кадре, не-человек, один снимок, нужен мгновенный
результат, упомянута не-Soul модель.

**Soul — шаги:**
1. Собрать 5–20 фото → media_upload/confirm (или image-job id / https URL, не локальные пути).
2. `show_characters(action:'train', name:…, images:[…])` — виджет сам поллит (~10 мин).
3. Готовность: `show_characters(action:'status', soul_id:…)` до `ready`.
4. Генерация: `generate_image(model:'soul_2', params:{soul_id, prompt, aspect_ratio})`;
   кинокадр → `soul_cinema_studio`.

**Elements — шаги:**
1. `show_reference_elements(action:'create', …)` из снимка → получить элемент(ы).
2. В промпте расставить плейсхолдеры `<<<UUID>>>` (можно несколько субъектов).
3. Генерация совместимой моделью (Nano Banana Pro / Seedream / Kling / Seedance).

**Дальше:** персонажа гнать в видео (`seedance_2_0`/`kling3_0`), оживлять
`motion_control`, апскейлить `upscale_*`.
