# Рецепт: раскадровка → по-сценные промпты Seedance 2.0

**Задача:** из сценария/идеи собрать шотлист и промпты для кинематографичного ролика
(реклама, короткометражка, демо) на `seedance_2_0`. Логика скила
`higgsfield-seedance-shotlist-director` (см. [урок](../lessons/2026-07-ai-ads-seedance.md)).

**Вход:** сценарий/скрипт + реф-ассеты (персонажи, продукт, локации, пропсы).

## Шаги
1. **Ассеты (стадия 1):** сгенерить реф-листы на нейтрально-сером фоне —
   `gpt_image_2` (product-sheets/схемы), `soul_cinematic` + Cinematic Locations
   (персонажи/локации). Одно лицо на лист; варианты одежды — отдельными листами.
   Загрузить в Elements, имена — говорящие (`@hero`, `@boss`, `@headphones`).
2. **Style-prefix (стадия 2):** написать единый префикс для всех сцен — объектив,
   модель света, `180° shutter motion blur`, детализация кожи, режиссура игры.
3. **По-сценные блоки:** для каждой склейки блок с именем (1a, 1b, 2a…):
   `Style header + Characters [@… с точными спеками] + Scene [локация, позиции пропсов,
   угол солнца, тени] + Cut-by-cut [камера: «29° FOV, 50mm, tracking, 180° shutter»;
   действие по битам; реплики/SFX; физ. блокинг]`.
   Continuity-lock на продукт: «@product stays locked rock-solid … through every shot».
   Аудио: «Diegetic dialogue and environmental SFX only. No music.»
4. **Генерация (стадия 3):** `generate_video(model:'seedance_2_0', params:{prompt:<блок>,
   aspect_ratio:'16:9', medias:[роли из Elements/референсов]})`. Держать 24fps, цель 4K/8K.
   Идентичность лица — через `seedance_2_0`; для одинаковых имён Elements подцепятся сами.
   `get_cost:true` перед пачкой.
5. **Сборка:** match-cut на одинаковом действии между сценами; при дрейфе персонажа —
   отдельные залоченные реф-ы (пот/усталость). Монтаж/музыка — отдельно (можно `/reel-factory`).

**Проверка:** лицо и продукт не «плывут» между сценами; камера-спеки соблюдены;
нет вшитой музыки. Опц. `virality_predictor` по финальному ролику.
