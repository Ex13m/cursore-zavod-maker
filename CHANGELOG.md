# Changelog

Формат — [Keep a Changelog](https://keepachangelog.com/ru/1.1.0/), версии — [SemVer](https://semver.org/lang/ru/).
Версия завода = версия корневого пакета; библиотека `cursorfx` версионируется отдельно.

## [1.3.0] — 2026-07-08

### Added
- **7 сцен, меняющих пространство** (cursorfx 0.4.0), собраны по анализу
  бестселлеров маркетплейсов + идеям владельца:
  `DogfightCursor` — космический бой: эскадрилья гонится за курсором-истребителем,
  промахи улетают за экран, остановка = сбит и взрыв, победный круг над обломками,
  движение = возрождение; стрельба по клику и сама по себе.
  `VortexCursor` — чёрная дыра, засасывающая частицы со всей страницы.
  `WarpCursor` — звёздный варп, скорость от движения мыши.
  `LightningCursor` — ветвящиеся молнии + полный разряд по клику.
  `RippleCursor` — сетка-ткань пространства, гнущаяся волнами.
  `RibbonCursor` — шёлковая лента (классический бестселлер, наша версия).
  `FireflyCursor` — живая стая светлячков, пугается рывков.

### Changed
- Дроп теперь ~70% живых/пространственных сцен; вес статичных иконок снижен.

### Removed
- Статичный спрайт-пак снят с производства (решение владельца: только динамика).

## [1.2.0] — 2026-07-08

### Added
- **Арт-спрайт-пак** (Higgsfield → remove background → 128×128 в `portal/public/sprites/`):
  Dragon Breath, Phoenix Dive, Voltage Skull, Iris Butterfly, Abduction UFO — каждый
  со своей парой эффектов (шлейф/свечение) и собственной идеей/ценой ($12).
- **Replicate-обогащение конвейера**: при `REPLICATE_API_TOKEN` фабрика ежедневно
  генерирует до `REPLICATE_DAILY_LIMIT` (3) свежих спрайтов вместо эмодзи; без ключа —
  тихая деградация.
- Превью карточки рисует настоящий спрайт (async-подгрузка картинки).

### Changed
- **Планировщик разнообразия**: взвешенная перетасовка архетипов без возврата —
  повтор идеи возможен только после использования всех (≈18) архетипов; пары
  «архетип+палитра» и имена внутри дропа не повторяются. Было: до 4 копий одного
  сценария на дроп; стало: максимум 2, и то в разных палитрах.

## [1.1.0] — 2026-07-08

### Added
- **Живые курсоры-сценарии** (cursorfx 0.3.0): `TurretCursor` (пушки бьют из-за кадра
  по перекрестью — кабина стрелка), `RocketCursor` (ракета преследует курсор и садится
  рядом, когда он замирает), `LureCursor` (воблер, за которым гоняется стая рыб),
  `NoiseBlobCursor` (пульсирующий абстрактный организм: шумовые 3D-конвульсии,
  периодическая смена цвета).
- **Генерируемый звук** (WebAudio, без файлов): выстрелы, гул двигателя, всплески,
  глитчи. По умолчанию выключен; тумблер 🔊 в карточке.
- **Рисунок курсора в карточке** — статичное превью до нажатия ТЕСТ.
- **Кнопка БРАК** — забраковать изделие (штамп, серость, снятие из очереди).
- **Анонс версии**: надписи о релизе вылетают из случайных взрывов и ~15 секунд
  летят на зрителя, плавно исчезая. Показывается один раз на версию.
- **Защита API**: ключ владельца `ZAVOD_ADMIN_KEY` (заголовок `X-Zavod-Key`) на
  одобрение/отгрузку/чат; кнопка КЛЮЧ на пульте.
- Документация: CHANGELOG, ROADMAP, SECURITY, ADR.

### Changed
- Дневной дроп: 10 → **20** курсоров (`DROP_SIZE` переопределяет).
- Vite 5 → 8, `@netlify/blobs` зафиксирован на 10.1.0 (audit: 8 → 1 low).

### Fixed
- Прод-функция `/api/publish` падала: конфликт `__dirname` при esbuild-бандлинге.
- `daily-factory`: `git pull --rebase` перед push (гонки с параллельными коммитами).
- Node 24 на раннерах Actions (Node 20 deprecated).
