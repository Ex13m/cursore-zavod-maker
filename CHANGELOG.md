# Changelog

Формат — [Keep a Changelog](https://keepachangelog.com/ru/1.1.0/), версии — [SemVer](https://semver.org/lang/ru/).
Версия завода = версия корневого пакета; библиотека `cursorfx` версионируется отдельно.

## [1.1.0] — 2026-07-08

### Added
- **Живые курсоры-сценарии** (cursorfx 0.3.0): `TurretCursor` (пушки бьют из-за кадра
  по перекрестью — кабина стрелка), `RocketCursor` (ракета преследует курсор и садится
  рядом, когда он замирает), `LureCursor` (воблер, за которым гоняется стая рыб),
  `NoiseBlobCursor` (пульсирующий абстрактный организм: шумовые 3D-конвульсии,
  периодическая смена цвета).
- **Генерируемый звук** (WebAudio, без файлов): выстрелы, гул двигателя, всплески,
  глитчи. По умолчанию выключен; тумблер 🔊 в карточке.
- **Рисунок курсора в карточке** — статичное превью до нажатия ТЕСТ.
- **Кнопка БРАК** — забраковать изделие (штамп, серость, снятие из очереди).
- **Анонс версии**: надписи о релизе вылетают из случайных взрывов и ~15 секунд
  летят на зрителя, плавно исчезая. Показывается один раз на версию.
- **Защита API**: ключ владельца `ZAVOD_ADMIN_KEY` (заголовок `X-Zavod-Key`) на
  одобрение/отгрузку/чат; кнопка КЛЮЧ на пульте.
- Документация: CHANGELOG, ROADMAP, SECURITY, ADR.

### Changed
- Дневной дроп: 10 → **20** курсоров (`DROP_SIZE` переопределяет).
- Vite 5 → 8, `@netlify/blobs` зафиксирован на 10.1.0 (audit: 8 → 1 low).

### Fixed
- Прод-функция `/api/publish` падала: конфликт `__dirname` при esbuild-бандлинге.
- `daily-factory`: `git pull --rebase` перед push (гонки с параллельными коммитами).
- Node 24 на раннерах Actions (Node 20 deprecated).
